# Requests

Place new feature requests in this directory.

Use `@request-author` to create or refine a request from a feature idea. The
request author should stop after the request is clear enough for
`@spec-reviewer`.

Use:

```text
specrepo/templates/feature-request.md
```

Recommended filename:

```text
YYYY-MM-DD-short-name.md
```

A request in this directory is not approval to implement. It is input for a spec
reviewer to create or update architecture first.
